# 💳 ATM PIN Checker

A simple Python program that checks ATM PIN authentication using basic conditional logic.

## 📌 Features
- Takes PIN input from user
- Checks if entered PIN is correct
- Displays access granted or denied message
- Beginner-friendly Python project

## 💻 Tech Used
- Python

## 🧠 Logic
If entered PIN matches stored PIN → Access Granted  
If not → Access Denied

## 🚀 How to Run
1. Open project in VS Code
2. Run the Python file
3. Enter your PIN when asked
4. Get authentication result
